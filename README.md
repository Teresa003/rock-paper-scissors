# Rock Paper Scissors

A simple browser-based Rock Paper Scissors game built with HTML, CSS, and JavaScript.

## Project Overview

This project is a lightweight web game where a user can play Rock Paper Scissors against the computer. It includes a score tracker and simple visual feedback for wins, losses, and draws.

## Creation Date

Created on September 14, 2020.

## Features

- Play against the computer
- Keep track of user and computer scores
- Simple and responsive game interface

## Getting Started

1. Open the project folder in your browser.
2. Load the main HTML file to start playing.

## Project Structure

- `page.html` - Main game page
- `page.css` - Styling for the game
- `page.js` - Game logic and score handling
